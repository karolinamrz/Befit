Befit
Befit to aplikacja webowa wspierająca zarządzanie aktywnością fizyczną. Projekt został przygotowany w ramach zaliczenia przedmiotu i pozwala na podstawową obsługę użytkowników, planowanie treningów oraz monitorowanie postępów.


Funkcjonalności
Logowanie i rejestracja użytkowników
Zarządzanie planem treningowym
Śledzenie aktywności i postępów użytkowników
